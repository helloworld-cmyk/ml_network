Bắt đầu ôn tập

Import file .md chứa câu hỏi trắc nghiệm hoặc tự luận. Tiến trình làm bài được lưu tự động cho từng file riêng biệt.

Câu 1: Nội dung câu hỏi img: assets/hinh-minh-hoa.png A. Đáp án A B. Đáp án B C. Đáp án C D. Đáp án D ans: B Câu 2: (Tự luận) Viết đáp án ans: nội dung đáp án | đáp án khác Câu 3: (Chọn nhiều đáp án) A. Đáp án A B. Đáp án B C. Đáp án C ans: A,C

Tự luận: không cần A/B/C/D, chỉ cần dòng ans:. Chấm linh hoạt (bỏ hoa/thường, khoảng trắng, dấu). Nhiều đáp án đúng (tự luận): dùng |. Chọn nhiều đáp án (trắc nghiệm): ans: A,C.

Ảnh: đặt file vào thư mục assets/ cạnh file .md, thêm dòng img: assets/ten-anh.png hoặc ![mô tả](assets/ten-anh.png)
